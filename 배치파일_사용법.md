# 배치 파일 사용법

프로젝트 뷰어/에디터를 쉽게 실행하고 관리할 수 있는 배치 파일들입니다.

## 📋 배치 파일 목록

### 1. `setup.bat` - 초기 설정
**처음 한 번만 실행**하세요.

```cmd
setup.bat
```

**기능:**
- Python 설치 확인
- 필수 패키지 설치
- LLM 제공자 선택 및 설치
  - Gemini (Google)
  - OpenAI (ChatGPT)
  - Anthropic (Claude)
- 기본 모듈 테스트

**언제 사용하나요?**
- 프로그램을 처음 설치할 때
- LLM 제공자를 추가하고 싶을 때

---

### 2. `test.bat` - 전체 테스트
프로그램이 올바르게 작동하는지 확인합니다.

```cmd
test.bat
```

**기능:**
- 구문 검사 (Syntax Check)
  - 모든 Python 파일의 문법 오류 확인
- Import 테스트
  - 모든 모듈이 올바르게 로드되는지 확인
- 프로젝트 구조 확인
  - 프로젝트 폴더 및 파일 존재 확인

**언제 사용하나요?**
- 코드를 수정한 후
- 프로그램 실행 전 문제 확인
- 오류가 발생했을 때 원인 파악

---

### 3. `run.bat` - 정상 실행
프로그램을 **상세한 확인 메시지와 함께** 실행합니다.

```cmd
run.bat
```

**기능:**
- Python 설치 확인
- Python 버전 표시
- 프로젝트 폴더 확인
- 프로그램 실행
- 오류 발생 시 메시지 표시

**특징:**
- 프로젝트 폴더가 없으면 경로 입력 요청
- 오류 발생 시 자동으로 일시정지
- 실행 전후 상태 확인

**언제 사용하나요?**
- 일반적인 프로그램 실행
- 문제 해결 중
- 자세한 실행 정보가 필요할 때

---

### 4. `start.bat` - 빠른 실행
프로그램을 **최소한의 확인 후 바로** 실행합니다.

```cmd
start.bat
```

**기능:**
- Python 설치 확인 (최소한의 안전장치)
- 프로젝트 경로 확인 (없으면 입력 요청)
- 즉시 프로그램 실행
- 오류 발생 시 일시정지

**언제 사용하나요?**
- 일상적인 빠른 실행
- 모든 게 정상적으로 작동하는 것을 확인한 후
- 바탕화면 바로가기로 사용

---

## 🚀 추천 사용 순서

### 처음 설치할 때:
```
1. setup.bat      (초기 설정)
2. test.bat       (테스트 실행)
3. run.bat        (첫 실행)
4. start.bat      (이후부터 빠른 실행)
```

### 일상적인 사용:
```
start.bat  (더블클릭으로 바로 실행)
```

### 문제가 생겼을 때:
```
1. test.bat       (문제 확인)
2. run.bat        (상세 정보 확인)
```

### 코드를 수정했을 때:
```
1. test.bat       (수정 사항 검증)
2. run.bat        (실행 확인)
```

## 💡 팁

### 배치 파일 실행 방법

#### 방법 1: 더블클릭 (권장)
- 파일 탐색기에서 배치 파일을 더블클릭하면 바로 실행됩니다.

#### 방법 2: 명령 프롬프트 (CMD)
```cmd
cd C:\NewProject\senior_contents\senior_project_manager\01_man\editors_app
start.bat
```

#### 방법 3: PowerShell
PowerShell에서는 현재 디렉토리의 스크립트 실행 시 `.\` 접두사가 필요합니다:
```powershell
cd C:\NewProject\senior_contents\senior_project_manager\01_man\editors_app
.\start.bat
```

또는 `cmd`를 통해 실행:
```powershell
cmd /c start.bat
```

### 바탕화면 바로가기 만들기
1. `start.bat` 파일을 우클릭
2. "바로가기 만들기" 선택
3. 바로가기를 바탕화면으로 이동
4. 더블클릭으로 바로 실행!

### 프로젝트 경로 변경
`run.bat`나 `start.bat` 파일을 편집기로 열어서:
```batch
python main.py --project ../001_gym_romance
```
이 부분의 `../001_gym_romance`를 원하는 경로로 변경하세요.

### LLM 제공자 추가 설치
`setup.bat`를 다시 실행하거나, 직접 명령어로 설치:
```cmd
pip install google-generativeai  # Gemini용
pip install openai                # OpenAI용
pip install anthropic             # Anthropic용
```

## ⚠️ 문제 해결

### "Python을 찾을 수 없습니다"
- Python이 설치되어 있지 않습니다
- https://www.python.org/downloads/ 에서 설치하세요
- 설치 시 "Add Python to PATH" 옵션을 꼭 체크하세요

### "tkinter를 찾을 수 없습니다"
- Python 설치 시 "tcl/tk and IDLE" 옵션이 선택되지 않았습니다
- Python을 재설치하거나 수정 설치를 통해 추가하세요

### "프로젝트 폴더를 찾을 수 없습니다"
- `run.bat` 실행 시 프로젝트 경로를 직접 입력하세요
- 또는 `run.bat` 파일을 편집하여 경로를 수정하세요

### PowerShell에서 "'start.bat' 용어가 인식되지 않습니다" 오류
PowerShell은 보안상의 이유로 현재 디렉토리의 스크립트를 직접 실행하지 않습니다.

**해결 방법:**
1. `.\start.bat` 형식으로 실행 (권장)
2. 또는 `cmd /c start.bat` 명령 사용
3. 또는 파일 탐색기에서 더블클릭 (가장 간단)

### 실행 시 오류 발생
1. `test.bat` 실행하여 어느 부분에서 문제가 있는지 확인
2. 오류 메시지를 확인하여 문제 해결
3. 필요시 `setup.bat`를 다시 실행
